# 教程 01: 快速入门

## 环境准备

### 系统要求
- Go 1.21+
- Node.js 18+
- pnpm（推荐）或 npm

### 推荐方式：一键启动脚本

最简单的方式是使用项目根目录下的跨平台启动脚本，它会自动完成前端构建、后端编译和服务启动。

**Windows:**
```powershell
# 双击运行
start.bat
# 或
start.ps1
```

**Linux / macOS:**
```bash
chmod +x start.sh
./start.sh
```

详细说明请参考 [教程 00: 跨平台一键启动](00-one-click-start.md)。

### 手动安装与启动

如果你需要手动控制构建过程，或进行开发调试，可以按以下步骤操作：

```bash
# 克隆项目
git clone <repo-url>
cd OverstepLab

# 单二进制运行
make build
./bin/oversteplab

# 或开发模式
# 终端1: cd src/backend && go run ./cmd/server/main.go
# 终端2: cd src/frontend && pnpm install && pnpm dev
```

启动后访问 `http://localhost:5000`。

## 界面概览

### 左侧导航栏
- **仪表盘**: 统计概览
- **VPS 管理**: 服务器列表和详情
- **企业成员**: (仅企业管理员可见)
- **订单/账单**: 财务管理
- **工单**: 技术支持
- **API Key**: API 访问凭证
- **审计日志**: 操作记录
- **漏洞挑战**: 核心功能入口

### 顶部栏
- **安全模式开关**: 在"漏洞模式"和"安全模式"之间切换
- **用户名**: 当前登录用户
- **退出按钮**: 登出
